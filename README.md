# Proyecto Farmacia

Proyecto de gestión básica de una farmacia: registro de clientes, gestión de
medicamentos (genéricos, de marca y controlados), ventas y persistencia
simple en archivo.

**Integrantes:**
- Julieta Fernandez
- Salvador Veron
- Agustin Heredia


**Descripción**:
- Programa en Java que permite agregar medicamentos, registrar clientes,
  vender medicamentos, reponer stock y mostrar reportes básicos.
- Persistencia simple mediante serialización a archivos (`clientes.dat`,
  `medicamentos.dat`) usando la clase `RepositorioArchivo`.
